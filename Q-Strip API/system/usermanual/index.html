1. Introduction

The Q-Strip 2.0 is a flexible, soft sensor measuring 90 cm in length and 3 cm in width. It is designed to be placed discreetly on the mattress, specifically at chest height, under the mattress sleeve. This device allows non-intrusive monitoring of night sweats and comes with a data logger for efficient data collection.

2. Package Contents

Your Q-Strip 2.0 package includes the following items:

  • 1 Q-Strip Data Logger V2.0
  • USB-A to USB-C power connector cable (2 meter)
  • 1 Q-Strip V1.04 sensor
  • 1 Micro SD card (pre-installed in the Q-Strip Data Logger)
  • Setup software for the Q-Strip Data Logger on the Micro SD card
  • Data analyzing software (for Windows) on the Micro SD card

3. Device Overview

Q-Strip Data Logger V2.0

  • Attached to the side of the mattress
  • Contains WiFi credentials, Home Assistant credentials, and Q-Strip public research API credentials stored on the Micro SD card
  • Operates with 5V USB power
  • Indicates the device status with different colored indicator lights
  • By default, data is stored on the local Micro SD card
  • Optional Your data can be send to a grouped (public) research into Your own cloud and Your homeassistant.

Q-Strip V1.04 Sensor

  • Flexible and non-invasive
  • Placed under the mattress sleeve at chest height for data collection

4. Setting Up the Q-Strip 2.0

Follow these steps to set up your Q-Strip 2.0:

  1. Q-Strip Data Logger Placement:
  • Attach the Q-Strip Data Logger to the side of the mattress.
  1. Q-Strip V1.04 Sensor Positioning:
  • Place the Q-Strip V1.04 sensor under the mattress sleeve at chest height.
  1. Micro SD Setup:
  • Insert the Micro SD card into a computer and run the setup software for configuring the Q-Strip Data Logger.
  • Follow the on-screen instructions to set up the device.
  • Do not forget to SET Your timezone.The device will update it’s own Real Time Clock by a positive wifi connection contacting NTP (Network Time Protocol).
  1. Data Logger Activation:
  • Once setup is complete, reinsert the Micro SD card into the Q-Strip Data Logger.

Launch Q-Strip_Setup.exe from the Micro-SD card.

Fill in Your SSID Wifi credentials and WPA2 password

Optional: You could change the parameters of the sequence to register and log the event faster or slower.

You can use Wifi to correct Your timezone. In the first boot while making connection with the internet the device will sett itself to the current time zone time. This is to correct the on board Real Time Clock, that’s being used for the local micro sd card registration/log

If You’re using a HomeAssistant (open source home control), You can enter the adres endpoint, Accessory ID and home Assistant token in the fields of tab “Homeassistant”. You can create Your token in the user settings tab in Your homeassistant software.

Using the Public API or create Your own

Using Your own API or the Public API (if You got an invitation), You can use the urls that’s being given.

You setup Your own API by downloading the simple php scripts from www.research.q-strip.com Only need is a webserver with PHP & MySQL. The generated token web based could be used to group Your devices in the API user interface.

5. Understanding Indicator Lights

  • Green Light: Indicates the WiFi is on and operational.
  • Orange Light: Signals the device is in the process of connecting to WiFi.
  • Blue Light: Confirms successful WiFi Cloud connection.
  • Green Blinking Light: Shows the Micro SD card is logging and backing up data.
  • RED Blinking: There is NO Micro SD card present.

note: at first boot using WiFi, it possibly could take a minute before the indicator lights up.

6. Device Functionality

Once set up, the Q-Strip 2.0 will continuously monitor night sweats discreetly. The device will log all data onto the Micro SD card for future analysis.

7. Data Management

To access and analyze the collected data:

  1. Insert the Micro SD card into a Windows computer.
  2. Use the provided data analyzing software (for Windows) to view and analyze the collected data.
  3. In case You registered The cloud API, Your data will be available online as well.Please check out more information at www.research.q-strip.com

  • Ensure the Q-Strip V1.04 sensor is placed securely under the mattress sleeve for optimal performance.
